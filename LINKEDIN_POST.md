🚀 **Excited to share my latest project: GateFlow!** 🏟️

Managing crowd flow in large venues is a massive logistical challenge. To tackle this, I built **GateFlow**—a real-time, smart stadium pedestrian routing and transit advising system! 

GateFlow calculates optimized, step-free paths for fan arrivals and post-match egress. It takes into account:
✅ Real-time gate congestion & wait times
✅ Accessibility requirements (e.g., wheelchair-friendly routes)
✅ Seating sections & kickoff urgency 
✅ Post-match transit options (Metro, Rideshare, Parking)

Instead of just showing a map, GateFlow acts as an **interactive AI copilot**. Under the hood, it uses a deterministic graph-routing engine (Dijkstra) paired with a **Google Gemini** phrasing layer to give fans natural, conversational directions in multiple languages (English, Spanish, French, Hindi).

Security and performance were top priorities:
🛡️ Built-in prompt injection sanitization
⚡ Token-bucket IP rate limiting with Redis fallback
📊 100% Code Coverage & strict CI/CD quality gates

I’m really proud of the architecture and how it balances algorithmic precision with a seamless, AI-driven user experience.
